Gameapalooza

Welcome to Team Bug Bashers' website entry to The Coding Olympics 2024 grand finale!

Features

  • Play games, view graphics, and more in 1 website! Advanced state system to allow for multiple programs to co-exist in 1 website, including an earlier version of this website called Library Admin App V1.
  • Would be over 8000 lines of code, but all programs by Team Bug Bashers' members here are in base64 for improved speed and performance. Without it, it this program would not fit on KA (it would not save and would break online editors such as JSBin and JSFiddle). Judges, if you want to see the performance difference, ask Liam K. to see an earlier version of this app (hosted on GitHub, use VSCode to view it). To see the code, click on the author links.
  • Finished August 26, 2024
  • Pure ES6 and CSS3 for the app
  • Extremely customizable CSS using root variables
  • 12 themes
  • Responsive navbar
  • Current theme gets saved to browser storage

Credit to YokieBob for converting some to Base64 and adding them to my program.
